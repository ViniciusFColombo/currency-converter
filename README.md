## 🇪🇸 Español

### Descripción
Un Dashboard financiero Full-Stack que ofrece conversión de divisas en tiempo real, análisis de tendencias históricas y un algoritmo de recomendación basado en lógica de mercado. Desarrollado para demostrar competencias en integración de APIs, visualización de datos y optimización de rendimiento.

### Características Principales
* **Conversión en Tiempo Real:** Tasas de cambio actualizadas mediante AwesomeAPI.
* **Gráficos Interactivos:** Seguimiento histórico de los últimos 7 días con **Chart.js**.
* **AI Insight:** Algoritmo personalizado que analiza la media semanal para sugerir señales de "Compra" o "Espera".
* **Optimización con Debounce:** Implementación técnica para minimizar las llamadas a la API y mejorar la respuesta de la interfaz.
* **Persistencia de Datos:** Historial de conversiones guardado localmente a través de LocalStorage.

### Tecnologías
* **Backend:** Python (FastAPI).
* **Frontend:** JavaScript (Vanilla), HTML5, CSS3.
* **Visualización:** Chart.js.

## 🇺🇸 English

### Description
A Full-Stack financial dashboard providing real-time currency conversion, historical trend analysis, and a market-logic-based recommendation algorithm. Built to showcase skills in API integration, data manipulation, and frontend performance optimization.

### Key Features
* **Real-time Conversion:** Live exchange rates via AwesomeAPI.
* **Interactive Charts:** 7-day historical price tracking using **Chart.js**.
* **AI Insight:** A custom algorithm that compares current prices with the weekly average to provide "Buy" or "Hold" signals.
* **Performance Optimization (Debounce):** Prevents excessive API requests by delaying the execution until the user stops typing.
* **Data Persistence:** Saves conversion history locally using LocalStorage.

### Tech Stack
* **Backend:** Python (FastAPI).
* **Frontend:** JavaScript (ES6+), HTML5, CSS3.
* **Visualization:** Chart.js.

## 🇧🇷 Português

### Descrição
Um Dashboard financeiro Full-Stack que oferece conversão de moedas em tempo real, análise de tendências históricas com gráficos interativos e um algoritmo de recomendação baseado em lógica de mercado. Desenvolvido para demonstrar habilidades em integração de APIs, manipulação de dados e otimização de performance.

### Principais Funcionalidades
* **Conversão em Tempo Real:** Integração com AwesomeAPI para taxas de câmbio atualizadas.
* **Gráficos Dinâmicos:** Visualização da variação dos últimos 7 dias utilizando **Chart.js**.
* **AI Insight:** Algoritmo personalizado que analisa a média semanal para sugerir momentos de compra ou venda.
* **Otimização com Debounce:** Técnica implementada para reduzir chamadas de API desnecessárias durante a digitação.
* **Histórico Persistente:** Uso de LocalStorage para manter as últimas conversões salvas no navegador.

### Tecnologias
* **Backend:** Python (FastAPI).
* **Frontend:** JavaScript (Vanilla), HTML5, CSS3.
* **Gráficos:** Chart.js.
